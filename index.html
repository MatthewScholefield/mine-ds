About

A 2D survival NDS game inspired by Minecraft. Explore uncharted lands across multiple biomes as you survive in this 2D free-roam game. Features include:

  • Inventory
  • Crafting
  • Zombies

How to get involved

  • Look at the issues page to see some items that need to get added
  • Setup the build environment
  • Look at some of the code related to that aspect
  • Message with any questions
  • Submit a pull request
